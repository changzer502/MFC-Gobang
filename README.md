# MFC-Gobang

使用MFC技术做五子棋游戏

* 用了mysql数据库做了玩家登录、注册和游戏战绩的记录
  * 文件中有sql文件
* 基本五子棋的实现
* 背景音乐切换
* 棋盘大小切换

(若要运行代码首先需要运行Gobang/data文件夹里的sql文件）
